# Reconnaissance
Reconnaissance is a process of gathering information about the target as much as possible it can be classified into two types:
- Passive Reconnaissance
- Active Reconnaissance

## Passive Reconnaissance
It is a type of scan which will not interact with the target , just collecting the information from the internet using various tools like Shodan, WHOIS, iplookup... and more.

## Active Reconnaissance 
It includes the attacker is interact with the target to gather informations using tools like Nmap, Sublister, Gobuster.....and many more.

### Passive Recon:
Tools Used:
- Shodan
- WHOIS
- whatweb
- wappalyzer

### Active Recon:
Tools Used:
- Nmap
- Feroxbuster
- Gobuster
